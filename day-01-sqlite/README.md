# Day 1 – SQLite Basics

## Overview
Today I started learning SQL using SQLite. I created a table, inserted data, and queried the data to answer simple business questions.

## What I Learned
- How to create a table
- How to insert data
- How to use SELECT
- How to filter using WHERE
- How to calculate values (revenue)

## Dataset Explanation
The dataset represents a simple sales table:
- product = item name
- price = cost per item
- quantity = number sold

## Key Insight
I calculated revenue using:
price × quantity

This is important because businesses use this to understand how much money each product generates.

## Takeaway
SQL is used to answer real business questions, not just to write queries.
